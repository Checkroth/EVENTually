About Our App

[EVENTually] is an event organization and hosting web solution. Currently, event hosting can be very difficult, and many events opt to simply use Facebook to do so. There are similar systems on the market, such as the Facebook event system, Eventbrite, etc. This system is meant to be dedicated strictly to event hosting, and contains many more tools than simply setting a calendar date and tracking invites.

Many events have different smaller, or sub-events within them such as speeches, tournaments, signings, etc. Our system is meant to be applicable to various types of events, so that users can utilize it for anything from planning a medical conference to a video game tournament.
